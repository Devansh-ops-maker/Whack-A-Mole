# Whack A Mole Game

This is a simple **Whack A Mole** game made using **Java**.  
The graphical user interface of the game is created using **Java Swing**.

The game displays a mole at random positions and the player has to click on it to score points.

---

## Technologies Used

- Java  
- Java Swing  

---

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Devansh-ops-maker/Whack-A-Mole.git
2. Open the project in any Java IDE like IntelliJ,Eclipse of NetBeans.
3. Compile and Run the file UIGame.java to play the game.
